Load<T>(String, Func<String, Boolean>, String[])

Loads all assets from the that match the given type paramter.

Declaration
public static List<T> Load<T>(string query = null, Func<string, bool> criteria = null, params string[] searchPaths)
    where T : UnityEngine.Object
Parameters
Type Name Description
String query

An optional query to use to filter assets. Providing a query will not immediately filter objects by type. The query can be any valid query from the projects tab.
Func<String, Boolean> criteria

An optional predicate to use to filter assets. It will be called with the path for each asset that matches the type or query given.
String[] searchPaths

An optional collection of the paths within the project to search. All paths should start with "Assets/".
Returns
Type Description
List<T>

A list containing the matching items or an empty list if no matches were found.
Type Parameters
Name Description
T

The Type of the assets to search for.
